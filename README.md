# lein-nevam

A Leiningen plugin to convert Maven pom.xml files to project.clj files.

## Usage

Put `[lein-nevam "0.1.0"]` into the `:plugins` vector of your
`:user` profile, or if you are on Leiningen 1.x do `lein plugin install
lein-nevam 0.1.0`.

    $ lein nevam
